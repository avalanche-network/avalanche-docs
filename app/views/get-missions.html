Mission List

GET /missions/{{test}}
Variables
count The number of items in the list
page The page of items to be return. Example: if count is 10 and page is 3, then the items 20-30 will be returned.
status Filter for the missions status. Status can be Planning, Launched, Completed or Failed, Canceled.
Description

You can get a list of missions and their details using this endpoint. Its also possible to do pagination and use filters to fine grain the returned results.

Examples
curl http://avalanche.network/api/missions/
